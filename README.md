# Компьютерная игра «Лабиринт Минотавра»

## Описание
Программа состоит из двух сцен: главное меню и игровая сцена. Меню имеет две кнопки: «ИГРАТЬ» для начала игры и «ВЫЙТИ» для выхода из игры. Игровая сцена представляет собой небольшой игровой мир, ограниченный лабиринтом, в котором находятся игровые предметы (Золотое руно, вазы со Священной амброзией и ядовитые змеи) и главный враг героя – Минотавр. Задача игрока: управляя персонажем, необходимо найти Золотое руно и выйти из лабиринта, собирая вазы с амброзией и избегая змей и Минотавра.
Продукт предназначен для повседневного использования любыми пользователями, которые хотят отдохнуть и развлечься, а также поупражнять своё мышление и внимание. Это могут быть люди разных возрастов и социального положения. 
Аналоги данного продукта мне неизвестны, а компьютерные игры подобного жанра довольно популярны, поэтому разработка данного приложения вполне актуальна.
## Установка и удаление
Перед началом работы с приложением «Лабиринт Минотавра» необходимо выполнить следующие действия:
1)	Открыть веб-браузер.
2)	Ввести в поисковую строку браузера ссылку на скачивание приложения: https://drive.google.com/file/d/17q_lAtgObJ-G5M_k1sf_kDeaM0C4dz_h/view?usp=drive_link (Windows), https://drive.google.com/file/d/1MezftyE6y8Eh1PBDk1eyybKKqiijRLrA/view?usp=drive_link (MacOS).
3)	На открывшейся странице Яндекс Диск или на странице Google Диск скачать архив программы.
4)	Разархивировать архив.
5)	Запускать приложение с файла MinotaurLabyrinth.exe.
6)	Также ознакомиться с основными рабочими файлами приложения можно по ссылке на репозиторий GitHub: https://github.com/Hypaspistai/MinotaurLabyrinth/tree/main .
7)	Для удаления игры необходимо удалить её корневую папку, распакованную из архива.
## Использование
1)	При запуске игры посредством двойного клика по её иконке пользователь попадает в главное меню, где расположены две кнопки: «ИГРАТЬ» и «ВЫЙТИ».
2)	Для начала игры необходимо кликнуть по кнопке «ИГРАТЬ» – будет загружен игровой мир, представляющий собой лабиринт с декоративным окружением. В верхнем левом углу расположена красная панель состояния здоровья, показывающая текущий уровень здоровья (количество очков здоровья) персонажа – героя Тесона, управляемого игроком. Управление стандартное: движениями мыши игрок может осматривать игровой мир; клавишами W (вперёд), A (влево), S (назад), D (вправо) игрок управляет движениями героя. Вернуться в меню можно, нажав клавишу ESC, но результат игры сохранён не будет. Герою необходимо найти в лабиринте Золотое руно.
3)	По мере продвижения по лабиринту герою будут встречаться игровые предметы, с которыми можно взаимодействовать («столкнувшись» с ними): вазы со Священной амброзией, ядовитые змеи и Золотое руно.
Вазы со Священной амброзией лечат героя, добавляя ему +20 единиц здоровья. Ядовитые змеи кусают героя, отнимая у него -20 единиц здоровья. Золотое руно – главная цель героя. Ему необходимо найти в лабиринте руно, забрать его и выйти из лабиринта – только в таком случае игра будет окончена победой. Без Золотого руна выйти из лабиринта невозможно! Главной опасностью для героя является Минотавр, бродящий по бесконечным коридорам лабиринта. Минотавр преследует героя с первых секунд начала игры и никогда не отстаёт! Его невозможно как-либо остановить, он не оставит героя в покое, поэтому игроку нельзя останавливаться. Атака Минотавра отнимает -100 единиц здоровья, так что каждый удар может стать последним. Игрок потерпит поражение, если герой потеряет все очки здоровья от Минотавра или змей. 
4)	Чтобы одержать победу, игроку необходимо выйти из лабиринта с Золотым руном. О наличии руна у героя свидетельствует зелёная панель с сообщением «Руно взято» в верхнем правом углу; она появляется, когда герой забирает руно.
5)	В случае победы (выхода из лабиринта с Золотым руном) или поражения (гибели героя) происходит выход в главное меню игры. Для выхода из игры необходимо кликнуть по кнопке «ВЫЙТИ».
